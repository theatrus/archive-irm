		BulkQueries (line 257)
	
 
	
	

Run an array of pre-specified queries  Churn through a pile of queries given in $querylist.


Special feature: if an entry in $queryset is an array, it will be  considered to be an associative array of dbtype => query entries.  We will run the query corresponding to the dbtype of the database  if possible, or otherwise an entry with the special dbtype of  '_default', if available.
Returns an array of failed queries, if such a thing is desired.  An empty array returned means "All good".
